# Installation
```shell
cd backend && node install
cd ../frontend && node install
```
_If chokidar errors, try `npm update --force`_
# Run

## Node Backend
```shell
cd backend
node server
```

## React Frontend
```shell
cd frontend
npm start
```

### Cypress
You can start Cypress with
```shell
npx cypress open
```

# GraphQL
You can access Apollo Playground at localhost:5000/graphql
